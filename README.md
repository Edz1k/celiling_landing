Натяжные потолки — современный лендинг для компании по установке потолков

Современный, быстрый и адаптивный лендинг, разработанный для компаний, занимающихся установкой натяжных потолков. Сайт помогает привлекать новых клиентов, демонстрировать выполненные проекты и упрощает процесс получения заявок через удобные формы связи.

О проекте

Проект создан с акцентом на конверсию посетителей в клиентов. На сайте представлены услуги компании, преимущества сотрудничества, портфолио выполненных работ, отзывы клиентов и удобные способы связи для быстрого оформления заявки.

Лендинг сочетает современный дизайн, высокую производительность и SEO-оптимизацию, что позволяет эффективно продвигать услуги компании в интернете.

Установка и запуск

Клонируйте репозиторий:

git clone https://github.com/Edz1k/celiling_landing.git

Перейдите в папку проекта:

cd celiling_landing

Установите зависимости:

pnpm install

Запустите проект в режиме разработки:

pnpm dev

После запуска приложение будет доступно по локальному адресу, указанному в терминале.

Сборка проекта

Для создания production-версии выполните команду:

pnpm build

Собранные файлы будут доступны в директории:

dist/

Структура проекта

src/
├── assets/ # Изображения и статические ресурсы
├── components/ # Переиспользуемые компоненты
├── layouts/ # Макеты страниц
├── pages/ # Страницы сайта
├── router/ # Настройки маршрутизации
├── styles/ # Глобальные стили
└── App.vue # Корневой компонент
