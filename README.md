⸻

📖 О проекте

Проект разработан для привлечения клиентов и удобного представления услуг компании. На сайте размещена информация о видах натяжных потолков, преимуществах сотрудничества, примерах выполненных работ, отзывах клиентов и способах связи.

Лендинг сочетает современный дизайн, высокую производительность и SEO-оптимизацию, помогая эффективно продвигать услуги компании в интернете.

🚀 Установка и запуск

Клонируйте репозиторий:

git clone https://github.com/Edz1k/celiling_landing.git

Перейдите в папку проекта:

cd celiling_landing

Установите зависимости:

pnpm install

Запустите проект в режиме разработки:

pnpm dev

После запуска приложение будет доступно по адресу, указанному в терминале.

⸻

📦 Сборка проекта

Для создания production-сборки выполните:

pnpm build

Готовые файлы будут находиться в директории:

dist/

### Настройка отправки заявок в Telegram

Добавьте в Netlify в разделе Environment variables:

```text
TELEGRAM_BOT_TOKEN=токен Telegram-бота
TELEGRAM_CHAT_ID=ID чата для заявок
```

После изменения переменных запустите новый deploy. Токен используется только Netlify Function и не попадает в клиентскую сборку.
