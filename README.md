Натяжные потолки

Современный лендинг для компании, специализирующейся на установке натяжных потолков.

О проекте

Сайт разработан для привлечения клиентов и удобного представления услуг компании. На лендинге размещена информация о видах натяжных потолков, преимуществах сотрудничества, примерах выполненных работ, отзывах клиентов и способах связи.

Основные возможности

- Адаптивный дизайн для смартфонов, планшетов и компьютеров
- Высокая скорость загрузки страниц
- SEO-оптимизация для поисковых систем
- Форма обратной связи
- Быстрый переход в WhatsApp и по телефону
- Галерея выполненных работ
- Информация об услугах и стоимости
- Современный и удобный пользовательский интерфейс

Технологии

- Vue 3
- Vite
- TypeScript
- UnoCSS
- Vue Router

Установка проекта

git clone https://github.com/Edz1k/celiling_landing.git
cd celiling_landing
pnpm install
pnpm dev

Сборка проекта

pnpm build

Готовые файлы будут находиться в папке dist.

SEO

Проект оптимизирован под поисковые запросы, связанные с натяжными потолками:

- Натяжные потолки
- Установка натяжных потолков
- Натяжные потолки цена
- Матовые потолки
- Глянцевые потолки
- Теневые потолки
- Парящие потолки
- Натяжные потолки под ключ

Цель проекта

Предоставить потенциальным клиентам удобный способ ознакомиться с услугами компании и быстро оставить заявку на расчет стоимости или консультацию.

Лицензия

Проект разработан для коммерческого использования компаниями, работающими в сфере установки натяжных потолков.
